给定一个二叉树的 **根节点** `root` ，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

 

 **示例 1:** 

<img src="https://assets.leetcode.com/uploads/2021/02/14/tree.jpg" style="width: 270px; " />

```

输入: [1,2,3,null,5,null,4]
输出: [1,3,4]

```
 **示例 2:** 

```

输入: [1,null,3]
输出: [1,3]

```
 **示例 3:** 

```

输入: []
输出: []

```
 

 **提示:** 
- 二叉树的节点个数的范围是 `[0,100]` 
- <meta charset="UTF-8" /> `-100 <= Node.val <= 100`  
 
**标签**
`树` `深度优先搜索` `广度优先搜索` `二叉树`
