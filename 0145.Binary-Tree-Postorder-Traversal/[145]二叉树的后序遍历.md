给你一棵二叉树的根节点 `root` ，返回其节点值的 **后序遍历** 。

 

 **示例 1：** 
<img alt="" src="https://assets.leetcode.com/uploads/2020/08/28/pre1.jpg" style="width: 127px; height: 200px;" />
```

输入：root = [1,null,2,3]
输出：[3,2,1]

```
 **示例 2：** 

```

输入：root = []
输出：[]

```
 **示例 3：** 

```

输入：root = [1]
输出：[1]

```
 

 **提示：** 
- 树中节点的数目在范围 `[0, 100]` 内
-  `-100 <= Node.val <= 100` 
 

 **进阶：** 递归算法很简单，你可以通过迭代算法完成吗？

 
**标签**
`栈` `树` `深度优先搜索` `二叉树`
